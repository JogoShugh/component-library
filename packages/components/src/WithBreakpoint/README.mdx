# WithBreakpoint

Provides the inner component with the breakpoint based on the Viewport width.

## Breakpoints

| Breakpoint | Use Case         |
| :--------- | :--------------- |
| **xs**     | Phone            |
| **sm**     | Tablet portrait  |
| **md**     | Tablet landscape |
| **lg**     | Desktop          |
| **xl**     | Large Desktop    |

<Playground>
  <WithBreakpoint>{width => <div>{width}</div>}</WithBreakpoint>
</Playground>
